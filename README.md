# 📋 ИС "Реестр предписаний"

Информационная система для Комитета образования Новгородской области по управлению предписаниями от контрольных органов.

## 🚀 Запуск

```bash
# Установить зависимости
pip install -r requirements.txt

# Запустить сервер
python manage.py runserver
```

Доступно на **http://localhost:8000**

Вход: **Login** → главное меню

## 🏗️ Структура проекта

```
apps/
├── accounts/         - Аутентификация
├── institutions/     - Учреждения
├── prescriptions/    - Предписания
├── funding/          - Финансирование
├── notifications/    - Уведомления
├── audit/            - Аудит
└── dashboard/        - Дашборды и UI
```

## 📊 Функционал

- ✅ Вход/выход (login/logout)
- ✅ Дашборд Комитета (статистика по 22 учреждениям)
- ✅ Дашборд учреждения (предписания)
- ✅ Мок-данные (22 учреждения, 24 предписания)
- ✅ Визуализация статусов (зелёный/жёлтый/красный)
- ✅ Отслеживание сроков

## 🔧 БД

PostgreSQL: `minobr` (credentials в `config/settings.py`)
